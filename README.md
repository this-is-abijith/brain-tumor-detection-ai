# 🧠 Brain Tumor Detection AI

A comparative machine learning and deep learning system for brain tumor detection using MRI images.

## 🚀 Models Used

- 🧠 MobileNetV2 (Transfer Learning CNN) – 92% Accuracy
- 📊 SVM with HOG Features – 86% Accuracy

## 📌 Features

- Dual model comparison
- Real-time prediction API (Flask)
- React frontend dashboard
- Accuracy comparison charts
- Grayscale MRI validation
- Prediction time measurement

## 🛠 Tech Stack

- Python
- Flask
- TensorFlow / Keras
- Scikit-learn
- React
- Chart.js
